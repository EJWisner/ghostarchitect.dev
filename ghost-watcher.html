How it works

Commit. Analyze. Fix. Repeat.

Ghost Watcher™ runs inside GitHub Actions. Every push triggers the full pipeline automatically.

01 — DETECT

Changed file detection

Ghost identifies exactly which files changed in the commit. Only changed files are primary targets — but the full codebase is loaded for context.

02 — ANALYZE

Blast Radius Analysis

Which files does this change affect? Ghost maps the dependency graph, rollback complexity, and risk tier for every changed file.

03 — DETECT

Conflict Detection

Contract mismatches, schema conflicts, and config errors between changed files and the rest of the codebase. The bugs that compile cleanly and surface in production.

04 — BRIEF

Ghost Brief™ generation

Every finding gets a structured, blast-radius-aware prompt your AI coding agent can execute immediately. Ghost briefs the AI. The AI fixes the code.

05 — DELIVER

Ghost Portal

Findings and prompts appear in Ghost Portal under the Ghost Watcher™ tab. Commits, Findings, and Prompts — with one-click copy to clipboard.

06 — COMMENT

PR comment

On pull requests, Ghost Watcher™ posts a findings summary directly on the PR — Phase 1 and Phase 2 breakdown — before the reviewer opens the tab.

Setup

Four steps. Ten minutes.

The Enable Watch wizard handles the hard parts. You add four secrets. Ghost Watcher™ fires on the next push.

1

Run the Enable Watch wizard

Open ghost in your terminal, scroll to Ghost Watcher™ → Enable Watch. The wizard asks which repo, which branches, which scans, and shows a cost estimate before you confirm. Ghost pushes ghost-watcher.yaml and the GitHub Actions workflow automatically.

2

Add four secrets to your GitHub repo

Go to Settings → Secrets and variables → Actions and add: ANTHROPIC_API_KEY, GHOST_LICENSE_KEY, GHOST_PORTAL_REPO, and GHOST_PORTAL_TOKEN.

3

Push a commit

Ghost Watcher™ fires on the next push to any watched branch. Watch the run in your repo's Actions tab. Results appear in Ghost Portal within minutes.

4

Open Ghost Portal and copy prompts

Ghost Portal → Ghost Watcher™ → Prompts tab. Every Ghost Brief™ prompt is ready to copy. Paste into Claude Code, Cursor, or any AI coding tool. Ship the fix. Push again. The loop tightens.
