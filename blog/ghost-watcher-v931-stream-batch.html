Ghost Architect™ now shows you the cost before every scan. Stream for real-time results, or submit as a batch and save 50% on API charges. Same analysis, same report, you decide.

What Changed

Every scan mode that calls the Anthropic API now opens with a transport menu before committing any spend.

New Commands

1
ghost batch-status

Check the status of any pending batch scans.

2
ghost batch-retrieve <id>

Pull results when ready. Delivers the same report files as streaming.

Pending batches appear in the main Ghost menu automatically when ready. Select to retrieve.

Ghost Watcher Now Sends Email

v9.3.1 adds two email notifications to Ghost Watcher.

Findings email: When Ghost Watcher detects issues, it emails you a full report -- severity summary, per-finding title, files affected, and detail. Link to your Ghost Portal for the complete scan history.
Clean scan email: When Ghost Watcher finds zero issues, it confirms the commit is clean. Your codebase is healthy. No action required.

Both are fire-and-forget. They never block the CI pipeline.
