What Ghost Brief™ Does

Every Ghost scan finds problems. Points of interest, blast radius, conflicts, fix forecasts. Ghost has always been exceptional at telling you what is wrong and where.

Ghost Brief™ closes the loop.

After any scan, run one command:

ghost --brief

Or skip the flag entirely. Ghost Brief™ is built into the interactive menu. After every scan, select Ghost Brief™ from the menu. Ghost auto-detects your most recent findings file and confirms before generating.

Either way, one step from scan to prompt pack.

Ghost reads your findings and converts them into a validated, blast-radius-aware prompt pack. A ghost-brief.json file your AI coding agent executes immediately. Each prompt tells the agent what to fix, which files to touch, which files to leave alone, and how to verify the fix landed correctly.

Ghost finds it. Ghost tells the AI exactly what to do. The AI does it.

Why Blast Radius Matters

Most AI coding agents are eager. They will fix what you point them at and break three things you did not know were connected.

Ghost Brief™ sorts every prompt by blast radius. Surgical fixes first. Broad changes last. The agent works in the safest possible order. You control what moves and when.

This is not vibe coding. This is governed AI remediation.

Works With Any Agent

ghost-brief.json is a plain JSON file. Feed it to Claude Code, Cursor, GitHub Copilot, or any other AI coding agent. Ghost does not lock you into a platform.

That said, Claude Code is the primary target. Anthropic controls half the AI coding market. Ghost Brief™ was built and tested against Claude Code first.

Ghost Portal™ Brief Tab

Every Ghost Brief™ you generate pushes automatically to Ghost Portal™. The Brief tab shows you every prompt, every blast radius score, every validation hint, and a full history of every Brief you have ever generated.

Your client does not need to understand the JSON. You do. The Portal is your command center.
