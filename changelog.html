Release notes.
What shipped, and when.

Ghost Architect™ ships fast. This page is the official record of what landed in each release, in plain language, with the user-facing change first. For deeper context on individual releases, see the corresponding posts on the blog.

We ship. We listen. We improve. Repeat.

Ghost Architect™ is not a launch-and-forget product. Every version below represents a real response -- to a gap we found, a customer need we heard, or a standard we weren't yet meeting. Ghost Watcher™ runs on every commit we make. We hold ourselves to the same bar we set for your codebase. This changelog is not a list of patches. It is a record of a product that takes its own advice.

Ghost scanned its own codebase. Found 7 issues. Fixed all 7. Zero findings on the next push.

  • ~High: Temp directory leak fixed — Commit Forecast now cleans up temp directories on all exit paths via try-finally. No more accumulated disk waste in CI environments.

  • ~High: Manifest concurrent write race fixed — Post-write verification now checks content, not just presence. Retry cycle includes exponential backoff.
  • ~High: Audit log failure threshold lowered — Warns on first failure (was third). Failure timestamp included in warning. On-disk marker file persists signal across CLI invocations.
  • ~High: File prioritizer error handling — Already fixed in dogfood pass 9. Confirmed present and passing.
