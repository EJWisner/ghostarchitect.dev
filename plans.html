// your code stays yours

Ghost runs locally. Your source code is never uploaded to our servers. BYOK — your Anthropic API key, your pipeline, your data.

// api data deleted in 7 days

Anthropic deletes API inputs and outputs within 7 days per their data retention policy. We have no access to that data.

// team sync stays private

Team and Enterprise sync data lives in your own private GitHub repo. Ghost Architect has no visibility into what you push or pull.

// cancel anytime

No contracts on Pro or Team. Cancel before your next billing date and you're done. Enterprise is governed by a formal MSA.
